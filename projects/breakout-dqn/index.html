Breakout-DQN

From-scratch Atari Breakout DQN study for building intuition around deep RL, preprocessing, replay, target networks, and early DeepMind-style agent design.

Implemented the learning loop to understand DQN mechanics from first principles.

proof
  • Frame preprocessing pipeline
  • Replay-buffer based value learning
  • Target-network training loop

Breakout-DQN is a deliberate first-principles RL project. The point is not to beat every public benchmark. The point is to understand the mechanics behind the early Atari agents: frame preprocessing, action repeat, replay buffers, target networks, value updates, and the instability that appears when any piece is wrong.

The project belongs in the portfolio because it shows the learning path behind the bigger robotics and world-model ambitions.
