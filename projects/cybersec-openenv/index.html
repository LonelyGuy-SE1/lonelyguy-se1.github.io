Cybersec OpenEnv asks whether a small local LLM can learn useful defensive behavior in a realistic long-horizon cyber environment. The setup models delayed alerts, partial observability, containment cost, adaptive attacker timing, and valid JSON actions.

The project became interesting when the model found reward exploits: isolate everything, or isolate the root asset at tick zero. The fix was not a bigger model. The fix was a better environment and an evidence-based containment reward that makes investigation before containment the winning strategy.

The held-out result matters most: the trained policy improved over the heuristic on a scenario it never saw during training. That suggests the model learned a transferable defensive pattern rather than memorizing attack graphs.
