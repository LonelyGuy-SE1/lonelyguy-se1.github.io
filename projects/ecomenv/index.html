EcomEnv is a smaller simulation project focused on the shape of agent environments: what the state exposes, what actions mean, how rewards can distort behavior, and how to make a toy system useful for learning.

It sits alongside the cyber and robotics work as practice in turning messy real-world workflows into controlled agent loops.
