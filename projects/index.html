home

Selected project case studies from SE1 across embodied AI, RL, LLM agents, vision, systems, and simulation.

  1. Raijin LOB phase one public

    C++ limit order book engine built around low-latency market microstructure ideas, benchmarked publicly and now moving toward a simulator phase.

  2. Project Halide public demo

    Small-model diagnostic workbench for damaged film scans using MiniCPM-V, Nemotron-Mini, strict validation, overlays, telemetry, and a Gradio light-table UI.

  3. Cybersec OpenEnv public research prototype

    Long-horizon cyber-defense POMDP for training and evaluating small LLM SOC agents with GRPO, realistic attack chains, and reward-hacking resistant containment incentives.

  4. Breakout-DQN learning project

    From-scratch Atari Breakout DQN study for building intuition around deep RL, preprocessing, replay, target networks, and early DeepMind-style agent design.
