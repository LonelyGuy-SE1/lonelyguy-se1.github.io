Project Halide is a diagnostic workbench for analog film scans. It does not try to magically restore the image. It extracts visible defect evidence, validates geometry, draws overlays, and writes a physical diagnosis that a human can inspect.

The important engineering choice is that the model is not treated as the final authority. MiniCPM-V proposes candidate defects, then a validator rejects malformed boxes, unsupported labels, edge artifacts, sprocket-lane noise, duplicates, and weak false positives before Nemotron-Mini writes the report.

The product lesson is simple: for messy real-world vision tasks, the validator and the UI are as important as the model. The interface exposes dropped boxes, edge artifacts, classical assist counts, raw JSON, and history so the user can judge uncertainty instead of being asked to trust a black box.
