Raijin is a fast limit order book engine built with consumer hardware in mind. The current public phase focuses on the core engine, benchmark reporting, and documentation around the design choices.

The project now moves into a simulator phase, where the goal is to model realistic market behaviour closely enough to support later RL and JEPA experiments.

The important part for the portfolio is the systems direction: market microstructure, latency-aware data structures, measured performance, and a path from engine work into agent simulation.
